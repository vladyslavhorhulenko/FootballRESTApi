This is the REST API service on Java Spring JPA with PostreSQL database.
Service provide all basic CRUD methods to managing football teams and players.
For testing how it works you should git clone this repo.
In project there are dockerFile and dokcer-compose.yml, but they not seems to working correctly, so
you should run application from folder target/. 
Press in cmd or shell "java -jar 
springjpapostgres-1.0.jar", wait for building and start to using rest api. 
You could test crud methods by the Postman, for example.
